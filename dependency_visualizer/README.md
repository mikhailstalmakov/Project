# depvis — этап 1: минимальный прототип

## 1. Общее описание
Минимальный CLI-прототип инструмента визуализации графа зависимостей.
На данном этапе программа читает XML-конфигурацию, валидирует параметры и выводит
все настраиваемые параметры в формате `ключ=значение`. Реализована обработка ошибок.

## 2. Описание функций и настроек
Файл: depvis_stage1.py

Настраиваемые параметры (через XML):
- package_name — имя анализируемого пакета.
- repo_url_or_path — URL репозитория или путь к файлу тестового репозитория.
- test_repo_mode — булево: `true`/`false` (если true — ожидается путь к файлу тестового репозитория).
- output_image_name — имя генерируемого файла с изображением графа (на данном этапе файл не создается).
- filter_substring — подпоследовательность для фильтрации пакетов (на этапе 1 только читается).

Файл конфигурации — пример `config.xml`.

## 3. Команды для сборки и запуска тестов
(Сборка не требуется — чистый скрипт Python.)

Пример запуска: python depvis_stage1.py --config config.xml

## 4. Примеры использования
См. `config.xml` — пример для рабочего режима (реальный URL) и пример для тестового режима (локальный файл).
